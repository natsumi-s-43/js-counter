# Counter App

A simple and interactive counter application built with **HTML, CSS, and JavaScript**.  
Users can increase, decrease, and reset the counter value through a clean and intuitive interface.

## Demo
https://natsumi-s-43.github.io/js-counter/

## Features

- Increment, decrement, and reset functionality
- Real-time counter updates
- Simple and responsive UI
- No external libraries or frameworks

## Technologies Used

- HTML
- CSS
- JavaScript

## Purpose

This project was created as a portfolio project to practice fundamental web development skills, including DOM manipulation and event handling using vanilla JavaScript.
